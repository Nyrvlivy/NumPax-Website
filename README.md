## Project Description

NumPax is a fintech web application designed to help users not only manage their personal finances effectively and accessibly but also to enhance their financial knowledge and skills. Built with popular and recognized technologies, NumPax integrates all aspects of a user's financial life ‚Äî from budgeting to investments ‚Äî into a single platform. It also provides a solid educational foundation, enabling users to make more informed choices and develop better long-term financial health, all within an intuitive and accessible system.

## How to Setup?

- [x] Fork this repository to your GitHub;
- [ ] Clone your repository to your machine;

```bash
    git clone https://github.com/Nyrvlivy/NumPax-Website
```

- [ ] Navigate to the project's root directory and open the **index.html** file in your browser.

```bash
    cd numpax
    open index.html
```

- [ ] Alternatively, you can view the deployed version [here]()
- [ ] And enjoy it! üéäüëè
